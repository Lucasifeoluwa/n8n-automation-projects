# n8n Automation Projects

## 1. AI Job Application Tracker
Automatically tracks job application emails using Gmail, Claude AI and Google Sheets.

**Tools used:** n8n, Claude AI (Anthropic), Gmail, Google Sheets

**How it works:**
- Monitors Gmail every minute for job emails
- Claude AI extracts company, job title, fit score and required skills
- Filters out non-job emails automatically
- Logs all job applications to Google Sheets

**File:** `AI_Job_Application_Tracker.json`
